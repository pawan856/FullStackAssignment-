# Full Stack Developer Pre-work Assignment

This repository contains the solution for the Full Stack Developer Pre-work Assignment.

## Structure

### Part 1: REST API (ASP.NET Core)
*   **Location**: `Part1_REST_API/`
*   **Description**: A basic ASP.NET Core Web API with CRUD operations, in-memory storage, and Basic Authentication.
*   **Instructions**: Navigate to `Part1_REST_API` and follow the instructions in its `README.md`.

### Part 2: Database Management (SQL)
*   **Location**: `Part2_Database/`
*   **Description**: SQL script (`OrderManagement.sql`) for creating the Order Management System schema, inserting data, and running analytical queries. Includes an ER Diagram in the `README.md`.
*   **Instructions**: Open `Part2_Database/OrderManagement.sql` in your SQL client (e.g., SSMS) and execute the script. View `Part2_Database/README.md` for design details.

### Part 3: Data Analysis (Python)
*   **Location**: `Part3_DataAnalysis/`
*   **Description**: Python data analysis using Pandas, Matplotlib, and Seaborn. Includes a mock dataset (`dataset.csv`) and a Jupyter Notebook (`Analysis.ipynb`).
*   **Instructions**: Navigate to `Part3_DataAnalysis` and follow the `README.md` to run the analysis.
